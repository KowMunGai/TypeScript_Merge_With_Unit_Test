# TypeScript merge with Unit test

## 1. Pull repository
    ```
    git clone https://github.com/KowMunGai/TypeScript_Merge_With_Unit_Test.git
    ```

## 2. Change directory to enter the project
    ```
    cd TypeScript_Merge_With_Unit_Test
    ```

## 3. Install depedencies
    ```
    npm i
    ```

## 4. Execute code
    ```
    npm test
    ```
